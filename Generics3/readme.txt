Deseja-se fazer um programa que leia uma quantidade N, e depois N números
inteiros. Ao final, imprima esses números de forma organizada conforme
exemplo. Em seguida, informar qual foi o primeiro valor informado.

1 - Leia um número N (quantidade de valores).
2 - Leia N números inteiros.
3 - Mostre todos os números organizados.
4 - Mostre qual foi o primeiro valor informado.
